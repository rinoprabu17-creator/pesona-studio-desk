# Mockup Generator Spec

## Tujuan

Membuat mockup awal yang cantik dan cepat untuk lead magnet. Mockup bukan desain final produksi.

## Batasan

- Tidak ada revisi mockup.
- Mockup awal tidak boleh dianggap file produksi.
- File final produksi tetap dibuat manual di Corel/PDF setelah cocok penawaran.
- Output diberi watermark halus: `Preview Mockup Awal`.

## Input minimal

```text
Nama Sekolah
Jenis Sampul: Raport / Ijazah
Warna: Auto / manual
Logo: Default / upload
Kota/Kabupaten opsional
```

## Template awal

1. Raport Maroon Formal
2. Raport Navy Formal
3. Raport Black Gold Premium
4. Raport Hijau Tua Formal
5. Ijazah Black Gold Formal
6. Ijazah Navy Elegant

## Warna awal

- Maroon
- Navy
- Hitam
- Hijau Tua
- Abu-abu
- Hijau Muda
- Biru Muda
- Coklat
- Orange

## Default warna berdasarkan jenjang

```text
SD  → Maroon
SMP → Navy
SMA → Hitam / Abu-abu
SMK → Navy / Hitam
MI  → Hijau Tua
MTs → Hijau Tua
MA  → Hijau Tua
```

## Default logo

Jika logo sekolah tidak ada:

- Tut Wuri Handayani
- Garuda
- Icon pendidikan
- Placeholder “LOGO SEKOLAH”

MVP tidak perlu otomatis mengambil logo kabupaten/kota dari internet. Gunakan asset library internal saja.

## Output

- PNG/JPG 1080x1350 untuk WA
- PNG/JPG 1080x1920 untuk story/status
- Link download/copy

## Rule layout

- Nama sekolah maksimal 2 baris.
- Kalau nama panjang, font otomatis mengecil.
- Logo default ditempatkan di tengah.
- Efek gold foil hanya simulasi visual.
- Tambahkan background menarik: meja, lighting, shadow, premium look.
- Jangan buat mockup terlalu mirip file final produksi.

## Copy sales setelah mockup dikirim

```text
Ini preview awal ya Pak/Bu, supaya Bapak/Ibu bisa membayangkan tampilan Sampul Raport/Ijazah sekolahnya.

Untuk desain final produksi, nanti tetap kami sesuaikan lagi berdasarkan logo, warna, bahan, ukuran, jumlah, dan data resmi sekolah setelah cocok penawaran.
```
